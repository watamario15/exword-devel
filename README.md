# CASIO EX-word DATAPLUS 5/6/7 Project Skeleton

This repository is a skeleton for developing homebrews for CASIO EX-word DATAPLUS 5/6/7. Just modify `src/main.c` and `make` to get build artifact in `build/ja/` and `build/cn/`.

- `html/`: Metadata HTML templates to be read by the EX-word system
- `src/`: C source files
